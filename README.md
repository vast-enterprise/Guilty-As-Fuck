## 项目简介

本项目是一款基于 **Unreal Engine** 开发的搞笑法庭辩论游戏。玩家扮演被告，与 AI 控制的法官、原告和证人等法庭角色展开荒诞而充满戏剧性的庭审交锋。

游戏的核心玩法在于：玩家可以通过 **Tripo AI** 实时生成 3D 证物模型，将其作为辩护证据呈堂，以增强辩词的可信度并争取辩论胜利。每一轮庭审都围绕非严肃、无厘头的案件展开，证物的生成结果与辩护策略的碰撞构成了游戏的核心乐趣。

## 使用的 Tripo 模型 / API

| 项目 | 说明 |
|------|------|
| **当前模型** | Tripo 1.4 —— 通过文本提示词（Text-to-3D）生成证物的 3D 模型 |
| **计划迁移** | Tripo Live —— 预计后续替换为该模型，以大幅提升生成速度，满足游戏内实时交互的节奏需求 |
| **调用方式** | 通过 Tripo API 在游戏运行时发起生成请求，获取 3D 模型资产并在 UE 场景中动态加载展示 |

## 当前限制 / 问题反馈

- **生成速度不足**：Tripo 1.4 模型的 Text-to-3D 生成耗时较长，无法匹配游戏内庭审辩论的实时交互节奏，玩家在等待证物生成期间体验存在明显中断感。
- **生成结果随机性较大**：部分提示词对应的 3D 模型输出结果一致性不高，相同或相似的描述可能产出差异显著的模型，影响玩家对证物呈堂效果的预期与策略制定。

> 以上两项限制预计在迁移至 **Tripo Live** 模型后得到显著改善。
